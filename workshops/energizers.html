⚡ WAKE-UP GAMES

// high energy · wake the group
G-01

ROBOT SAYS

Like Simon Says — but with robot commands.

▸ HOW TO PLAY

  1. Grown-up: "Robot says ROTATE!" → all group rotate
  2. "Robot says BEEP!" → all beep
  3. "Walk forward!" (no "Robot says"!) → who moves = OUT
  4. Last operator standing = victor
⏱ 3-5 min🔊🔊🔊👥 any size
EFFECT: listening + body movement = perfect wake-up
G-02

SIGNAL FREEZE

Music ON = move. Music OFF = STATUE.

▸ HOW TO PLAY

  1. Play any energetic track
  2. Group moves like field kids
  3. Pause music randomly → all FREEZE
  4. Whoever moves = sit out (or 5 push-ups)
⏱ 3 min🔊🔊🔊👥 5+
EFFECT: burns kinetic energy · sharpens reaction time
G-03

BATTERY RECHARGE

Group cardio framed as power restoration.

▸ HOW TO PLAY

  1. All standing. "Battery: 0%! [STATUS: CRITICAL]"
  2. 10 jumping jacks → "BATTERY: 25%"
  3. 10 squats → "BATTERY: 50%"
  4. 10 high-fives with neighbour → "BATTERY: 100% · OPERATIONAL"
⏱ 2 min🔊🔊👥 any
EFFECT: quick cardio · keeps the robot theme going
G-04

ROBOT WALK RELAY

Stiff-legged walk relay between teams.

▸ HOW TO PLAY

  1. 2 teams. Stiff "robot walk" only
  2. Tag teammate at end → next kid goes
  3. First team done = field champions
  4. Bonus: must beep at every turn
⏱ 5 min🔊🔊🔊👥 6+
EFFECT: cardio + competition + ridiculous walking
G-05

BEEP-BOOP TAG

Tag, but every tag requires vocal signal.

▸ HOW TO PLAY

  1. 1 operator is "active"
  2. Tag = touch + say "BEEP"
  3. Tagged operator becomes active + says "BOOP"
  4. Forgetting signal = stuck for 5 sec
⏱ 5 min🔊🔊🔊👥 6+
EFFECT: classic tag + cognitive overhead = harder than it looks
G-06

RECON HUNT

"Locate something RED in 30 sec! GO!"

▸ HOW TO PLAY

  1. Grown-up names a property: "round" / "blue" / "soft"
  2. 30-second timer
  3. Group scrambles to retrieve matching object
  4. Best find / fastest find = points
⏱ 3-5 min🔊🔊🔊👥 4+
EFFECT: sensors theme + scavenger hunt energy

🧠 BRAIN GAMES

// focus · thinking · stuck-buster
G-07

MISSING ASSET

Memory test with 8 robot components.

▸ HOW TO PLAY

  1. Lay 8 small objects on a tray (cap, pen, key…)
  2. Group observes for 30 sec, then closes eyes
  3. Remove ONE object
  4. Open eyes → first to identify missing item wins
⏱ 3 min🔊👥 any
EFFECT: resets attention · quiet but engaging
G-08

CHAIN BRIEFING

Each kid adds one sentence to a robot log.

▸ HOW TO PLAY

  1. Circle. Open with: "Robot ROBI-7 detected an unusual signal..."
  2. Next kid adds 1 sentence
  3. Continue around circle
  4. Stop when log = absurd (5 min max)
⏱ 5 min🔊🔊👥 5-12
EFFECT: listening + creativity + safe failure
G-09

ALGORITHM CHARADES

Act out an algorithm. Group guesses each step.

▸ HOW TO PLAY

  1. Volunteer picks card: "make sandwich", "brush teeth", "tie laces"
  2. Acts it STEP-BY-STEP, no speaking
  3. Group guesses each step
  4. Bonus: "what was step 4?" tests memory
⏱ 5 min🔊🔊👥 4+
EFFECT: reinforces "algorithm = ordered steps" theme
G-10

SIGNAL TELEPHONE

Broken-phone, but with movement sequences.

▸ HOW TO PLAY

  1. First operator does 3-move sequence (rotate, jump, clap)
  2. Shows next operator ONCE
  3. Each operator passes to next
  4. Last one performs → compare to original
⏱ 5 min🔊🔊👥 6+
EFFECT: memory + observation · hilarious distortion

🤝 TEAM GAMES

// cooperation · bonding · reset divisions
G-11

TACTICAL KNOT

Untangle yourselves without breaking formation.

▸ HOW TO PLAY

  1. Stand in circle. Reach across, grip 2 random hands
  2. Don't grip a neighbour
  3. Without breaking grip → untangle into circle
  4. Talk · plan · move slow
⏱ 5-10 min🔊🔊👥 6-12
EFFECT: forces cooperation + spatial reasoning
G-12

SHADOW PROTOCOL

Pair up. One leads, one mirrors EXACTLY.

▸ HOW TO PLAY

  1. Pairs face each other. NO talking
  2. Lead kid makes slow movements
  3. Shadow copies EXACTLY in real-time
  4. 30 sec → swap roles
⏱ 4 min🔊👥 pairs
EFFECT: builds focus + non-verbal sync
G-13

2 TRUTHS · 1 DECOY

3 robot-related statements. 2 true. 1 fake.

▸ HOW TO PLAY

  1. Each kid gives 3 robot statements
  2. 2 are TRUE · 1 is DECOY
  3. Group votes on the decoy
  4. Reveal · bonus point for fooling everyone
⏱ 5 min🔊🔊👥 4-10
EFFECT: sharing facts + reading bluffs = bonding
G-14

ASSEMBLY OPERATION

Stack 10 cups in 3 min using string ring.

▸ HOW TO PLAY

  1. Each team: 10 plastic cups + 1 rubber band with 4 strings
  2. Each kid holds 1 string
  3. Build pyramid using ONLY the strings
  4. No hands on cups
⏱ 8 min🔊🔊👥 4/team
EFFECT: pure cooperation · hilarious failures, satisfying wins

😄 LAUGH BREAKS

// quick laughs · break the tension
G-15

ROBOT IMPRESSION

Worst robot impression wins.

▸ HOW TO PLAY

  1. Each kid does best robot impression + voice
  2. One by one. Audience scores 1-10
  3. Most absurd = victor
  4. Bonus: grown-up goes too
⏱ 3 min🔊🔊🔊👥 any
EFFECT: self-deprecating laughs are the best
G-16

FREQUENCY CHOIR

Everyone outputs robot sounds simultaneously.

▸ HOW TO PLAY

  1. Form a circle
  2. Grown-up: "READY... BEEP!" → all beep in unison
  3. Then: "BOOP!" "ZAP!" "BEEEEP!"
  4. End with 30 sec ambient robot noise
⏱ 2 min🔊🔊🔊👥 4+
EFFECT: absurd · releases inhibitions
G-17

ROBOT-SENSOR-POWER

Rock-paper-scissors with robot themes.

▸ HOW TO PLAY

  1. ROBOT beats SENSOR (smashes it)
  2. SENSOR beats POWER (drains it)
  3. POWER beats ROBOT (powers off)
  4. Use poses · best of 5
⏱ 2 min🔊🔊👥 pairs
EFFECT: quick · themed · easy to learn
G-18

STARE PROTOCOL

Pair stares. First to break LOSES.

▸ HOW TO PLAY

  1. Pairs face each other 30 cm apart
  2. Stare into each other's eyes — NO breaking
  3. Audience can make weird faces in background
  4. Last pair to last 30 sec wins
⏱ 3 min🔊🔊👥 pairs
EFFECT: builds trust + uncontrollable laughter

🧘 CALM-DOWN GAMES

// after wild moments · reset focus
G-19

LOW-POWER MODE

Slow breathing as "robot powering down".

▸ HOW TO PLAY

  1. "Battery critical · conserving power"
  2. Inhale 4 sec → hold 4 sec → exhale 4 sec
  3. Repeat 5 cycles
  4. "System reboot complete"
⏱ 2 min🔇👥 any
EFFECT: box breathing · genuine calm-down science
G-20

STATUE STATE

Last kid to move WINS.

▸ HOW TO PLAY

  1. "Robots POWER OFF"
  2. Everyone freezes in their pose
  3. Grown-up times — must NOT move at all
  4. Last 3 standing get a sticker
⏱ 3 min🔇👥 any
EFFECT: forces stillness · builds patience
G-21

FOCUS CALIBRATION

Count clicks. Reset the cognitive stack.

▸ HOW TO PLAY

  1. Sit cross-legged, eyes closed
  2. Listen for grown-up clicker (5 clicks)
  3. Count silently: 1...2...3...4...5
  4. Open eyes → ready for next OP
⏱ 2 min🔇👥 any
EFFECT: mini-meditation · focused listening
G-22

SHUTDOWN SEQUENCE

Pretend to slowly power down body parts.

▸ HOW TO PLAY

  1. Standing. "Disengaging arms" → arms drop
  2. "Disengaging legs" → wobbly knees
  3. "Powering off head" → head drops
  4. Slowly settle to floor
⏱ 2 min🔊👥 any
EFFECT: body scan + relaxation in robot framing

+ BONUS DRILLS

// mix & match favourites
G-23

DECENTRALIZED COUNT

Group counts to 20 — no leader.

▸ HOW TO PLAY

  1. Group counts aloud: 1, 2, 3... up to 20
  2. No assigned order — anyone can call a number
  3. 2 operators speak at once → restart from 1
  4. Reach 20 = group cheer
⏱ 5 min🔊👥 4+
EFFECT: forces listening · group consciousness
G-24

RAPID DESIGN

90 seconds to sketch a custom robot.

▸ HOW TO PLAY

  1. Each kid: paper + pen
  2. 90 seconds. EXECUTE.
  3. Show all designs around the group
  4. Vote: most creative · most operational · weirdest
⏱ 3 min🔊👥 any
EFFECT: quick burst of creativity · easy success
G-25

ROTATION TEST

Spin in place. Then FREEZE on command.

▸ HOW TO PLAY

  1. All spin slowly on the spot
  2. "FREEZE" → all halt
  3. Whoever wobbles or falls = sit out
  4. Last kid standing wins
⏱ 3 min🔊🔊👥 any
EFFECT: cardio + balance + amusing failures
